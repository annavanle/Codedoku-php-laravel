## About Codedoku

Codedoku is a web application write by PHP and Laravel framework.
This web is a place to share code learing experience in Moblie, Web and others, from beginer to hero.

## Setting 
Codedoku is created by Laravel 5.3 and PHP 7.3 
The project can run with MAMP (MAC ) or XAMPP (windows)
